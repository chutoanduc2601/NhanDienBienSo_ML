# 🏍️ Nhận Diện Biển Số Xe Máy

Sử dụng YOLOv8 và EasyOCR để phát hiện xe và đọc biển số từ ảnh.

## 🚀 Cài đặt

```bash
git clone https://github.com/NDkhoa23/hocmay3.git
cd hocmay3
pip install -r requirements.txt
```
